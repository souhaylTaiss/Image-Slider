# Image Slider

An interactive 3D image carousel built with vanilla JavaScript, GSAP, and Tailwind CSS v4.

## Features

- 3D rotating carousel controlled by scroll, swipe, and keyboard
- Click on any item to expand it fullscreen with a smooth animation
- Animated navigation links using GSAP SplitText
- Entrance animations on page load
- Responsive design

## Tech Stack

- **Vite** — build tool
- **GSAP** — animations (SplitText, Observer)
- **Tailwind CSS v4** — styling

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
